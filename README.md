## Español

Bienvenido al repositorio oficial de **Pulsar**. Este proyecto ofrece soluciones elegantes para la descarga y procesamiento de contenido de YouTube.

### 📁 [Pulsar Desktop Suite](./Pulsar-Desktop)
Nuestra potente suite de escritorio para Windows. Incluye tres versiones (v1.0, v1.1, v1.2) que van desde una interfaz ligera hasta una experiencia web moderna en el escritorio.
- **Estado:** Estable / v1.2
- **Uso:** Descarga local, alta velocidad, múltiples formatos.

### 🌐 [Pulsar Web App (Próximamente)](./Pulsar-Web)
Versión en la nube basada en Streamlit, diseñada para ser ejecutada directamente desde el navegador (ideal para Hugging Face Spaces).
- **Estado:** En desarrollo (v1.3-web)

## English

Welcome to the official **Pulsar** repository. This project offers elegant solutions for downloading and processing YouTube content.

### 📁 [Pulsar Desktop Suite](./Pulsar-Desktop)
Our powerful desktop suite for Windows. It includes three versions (v1.0, v1.1, v1.2) ranging from a lightweight interface to a modern web experience on the desktop.
- **Status:** Stable / v1.2
- **Use:** Local download, high speed, multiple formats.

### 🌐 [Pulsar Web App (Coming Soon)](./Pulsar-Web)
Cloud version based on Streamlit, designed to be run directly from the browser (ideal for Hugging Face Spaces).
- **Status:** Under development (v1.3-web)
