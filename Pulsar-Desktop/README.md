# Pulsar - YouTube Audio & Video Suite

[Español](#español) | [English](#english)

---

## Español

**Pulsar** es una suite de herramientas diseñada para descargar y procesar contenido de YouTube de manera eficiente, elegante y sencilla. Este proyecto representa la evolución de un descargador simple a una aplicación robusta con una interfaz moderna.

### Versiones Incluidas

- **Pulsar v1.2 (Actual):** Esta es la versión principal alojada en la raíz. Interfaz Web moderna (Eel), previsualización de videos y descarga optimizada.
- **Versiones Anteriores:** Las versiones `v1.1` (CustomTkinter) y `v1.0` (Tkinter) se encuentran disponibles en la carpeta `archive/`.

### Instalación y Uso

Cada versión es independiente y contiene sus propios archivos de configuración:

1. Entra en la carpeta de la versión que quieras usar (ej. `v1.2`).
2. Ejecuta `install_requirements.bat` para configurar el entorno automáticamente (solo la primera vez).
3. Ejecuta `run_pulsar.bat` para iniciar el programa.

### Requisitos
- **Python 3.10+** instalado en el sistema.
- Conexión a internet.

## English

**Pulsar** is a suite of tools designed to download and process YouTube content efficiently, elegantly, and simply. This project represents the evolution from a simple downloader to a robust application with a modern interface.

### Included Versions

- **Pulsar v1.2 (Latest):** This is the main version hosted in the root. Modern Web interface (Eel), video preview, and optimized downloading.
- **Previous Versions:** Versions `v1.1` (CustomTkinter) and `v1.0` (Tkinter) are available in the `archive/` folder.

### Installation and Usage

Each version is independent and contains its own configuration files:

1. Enter the folder of the version you want to use (e.g., `v1.2`).
2. Run `install_requirements.bat` to automatically set up the environment (first time only).
3. Run `run_pulsar.bat` to start the program.

### Requirements
- **Python 3.10+** installed on the system.
- Internet connection.
